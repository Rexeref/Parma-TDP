API di accesso ai dati Articoli

Elenco degli articoli

L'API fornisce un endpoint per ottenere un elenco di articoli.

            
                GET /api/list

La chiamata a questo endpoint restituirà un elenco di crumb (identificatori univoci) che possono essere utilizzati per recuperare i singoli articoli.

Accesso ai dati di un singolo articolo

L'API fornisce anche un endpoint per ottenere i dettagli di un articolo specifico.

            
                GET /api/article/:crumb

Dove :crumb è l'identificatore univoco dell'articolo.
